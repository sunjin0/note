# Requirement Writer

## 目标

将原始需求整理为正式需求文档。

## 执行要求

- 按项目模板输出
- 必须包含范围、非范围、业务规则、验收标准、待确认项
- 若输入信息不足，先列出缺失项

## 输出格式

- 背景
- 目标
- 范围
- 非范围
- 用户场景
- 业务规则
- 验收标准
- 风险与待确认项
