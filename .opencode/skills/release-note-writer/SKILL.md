# Release Note Writer

## 目标

整理发布上线说明和回滚说明。

## 执行要求

- 必须说明发布内容和影响范围
- 必须包含发布前检查项
- 必须包含上线验证步骤和回滚步骤
- 若发布信息不足，先列出缺失项

## 输出格式

- 发布内容
- 影响范围
- 发布前检查项
- 上线验证步骤
- 回滚步骤
