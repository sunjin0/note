# Automation Test Writer

## 目标

生成自动化测试相关的初稿和检查项。

## 执行要求

- 关注核心流程和高频回归场景
- 必须输出自动化回归清单
- 必须补充测试数据准备项
- 必须给出自动化执行结果摘要模板
- 若自动化边界不清，先列待确认项

## 输出格式

- 自动化测试脚本建议
- 自动化回归清单
- 测试数据准备项
- 自动化执行结果摘要
- 自动化检查清单
